Setup

Connect your Mapbox tokens

Two tokens are needed — one for map tiles, one for the Feedback API. Both are stored only on this device and never leave it.

map
feedback

Map token (pk.*)

Your default public token from account.mapbox.com → Access Tokens. No special scopes needed — just copy the default pk.* token.

Feedback token (sk.*)

  1. 1 Go to account.mapbox.com → Access Tokens
  2. 2 Click + Create a token and name it CycleWatch
  3. 3 Under Secret scopes, check USER-FEEDBACK:READ and USER-FEEDBACK:WRITE
  4. 4 Click Create token and copy the sk.* value

Both tokens are stored in your browser's localStorage — they never leave this device.
